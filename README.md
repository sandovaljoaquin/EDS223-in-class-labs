# EDS 223 In Class Labs

Joaquin Sandoval 

Repository containing weekly labs in EDS 223.

Week 1: Explores the basics of map-making in R using the `tmap` package. 

Week 2: Intro to vector data with sf and coordinate reference systems

Week 3 : Basics of spatial and geometry operations on vector data in R using the sf package.

Week 4: Basics of raster data, including spatial data and geometry operations.

Week 5: Explore operations that rely on interactions between vector and raster datasets, including how to convert raster data into vector data.

Week 6: Operations that rely on interactions between vector and raster data. Convert raster data into vector data and create false color imagery.
